# 隐私说明

本应用（SGI）尊重用户隐私，收集和传输的数据范围如下：

## 更新检查

启动后自动向 GitHub API 发送请求检查新版本，传输内容：
- 当前应用版本号
- 请求来源 IP 地址（HTTP 协议本身携带）

不使用第三方统计或分析服务。

## 联机功能

本应用仅建立用户之间的点对点连接。所有游戏流量经由本地隧道直连或中继转发，应用本身不记录、不存储传输内容。

## 日志

日志仅存储在本地设备，用于排查连接问题，不会自动上传。

## 第三方链接

应用内的"意见反馈"和"开源地址"等链接会跳转到外部网站，这些网站的数据处理不受本隐私说明约束。
