# 更新日志

## [1.2.0] — 2026-07-07

### 新增
- 优化 UDP 打洞机制，支持双端同时打洞，提升了 NAT 穿透成功率，更多网络环境可实现直连
- Windows 版更新下载支持 Gitee 加速通道，提升国内用户下载体验

### 修复
- 修复断开联机时偶现的应用程序崩溃问题

### 优化
- 更新下载与安装流程的稳定性改进
- 中继服务器稳定性优化

---

## [1.1.0] — 2026-07-01

### 新增
- 日志自动清理：每次启动时自动删除旧的日志文件，避免占用过多磁盘空间。默认保留最近 7 天，可在设置中调整或关闭。

### 修复
- 修复了通过中继联网时，部分游戏（如 Minecraft）连接成功后数据不通的问题。

### 优化
- 提升了数据传输的稳定性，减少因数据包处理异常导致的连接问题。

---

## [1.0.0] — 2026-06-27

第一个正式版本。

### 功能
- 局域网游戏自动发现与检测（支持 Minecraft、Terraria、Stardew Valley 等）
- 创建/加入房间，通过互联网联机（中继转发 + 自动 NAT 打洞）
- IPv6 直连 / P2P 直连 / 中继转发多路径自动选择
- 网络环境检测（IPv6 可用性、NAT 类型）
- 自动更新检查与后台静默下载
- 多语言支持（中文 / English）
- 跨平台支持（Windows、macOS、Linux）
